# Spring-Chessgame

Dependencies: 
-----------------------------------------------------
- Apache Maven 3.5.0 
- Java version: 1.8.0_91

To Run: 
------------------------------------------------------
- Go to home directory: '/Spring-Chessgame'
- cmd: > mvn clean package
- Navigate to '/Spring-Chessgame/target'
- cmd: > java -cp DrawTest2-0.0.1-SNAPSHOT-jar-with-dependencies.jar main.Main
